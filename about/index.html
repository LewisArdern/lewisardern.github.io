About

I work as an application security consultant at Cigital, now part of Synopsys Software Integrity Group, based in San Francisco, CA.

My area of expertise is web application security, with a focus on client-side and JavaScript security. I enjoy creating and delivering security training to various types of organizations and institutes to help them understand security topics. I have worked part of a research team to identify security issues in frameworks such as Express, AngularJS and React.

As a security consultant I help integrate security into organizations SDLC processes, work part of internal security teams as an SME resource in regards to application security, and perform various penetration tests, mobile security assessments, and source code reviews.
